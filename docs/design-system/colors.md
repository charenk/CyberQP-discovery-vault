# Color System

## Brand Colors
- Primary: `brand.500` (#0073e6) - Main brand color for primary actions
- Secondary: `brand.600` (#005cb3) - Hover states and emphasis

## Semantic Colors
- Success: `green.500` - Successful operations
- Warning: `yellow.500` - Warnings and cautions
- Error: `red.500` - Errors and destructive actions
- Info: `blue.500` - Informational messages

## Neutral Colors
- Background: `gray.50` - Page backgrounds
- Surface: `white` - Card and container backgrounds
- Text Primary: `gray.900` - Main text
- Text Secondary: `gray.600` - Secondary text
