# Spacing System

## Overview
CyberQP uses Chakra UI's default spacing scale (4px base unit).

## Common Spacing Values
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## Usage Guidelines
- Use consistent spacing between related elements
- Maintain visual hierarchy through spacing
- Follow 8px grid system for alignment
