# Typography

## Font Families
- Heading: System font stack (San Francisco, Segoe UI, etc.)
- Body: System font stack

## Font Sizes
- Heading sizes: lg, md, sm (Chakra UI defaults)
- Body: 14px (0.875rem) base size
- Small text: 12px (0.75rem)

## Font Weights
- Headings: 600 (semibold)
- Body: 400 (regular)
- Emphasis: 600 (semibold)

## Line Heights
- Headings: Tight (1.2)
- Body: Comfortable (1.5)
