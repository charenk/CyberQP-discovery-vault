# CyberQP Documentation

This directory contains UX guidelines, design system documentation, and use cases for the CyberQP Privileged Access Management platform.

## Structure

- `ux-guidelines/` - Component guidelines, error states, and use cases
- `design-system/` - Colors, typography, spacing, and design tokens
- `patterns/` - Common UI patterns and workflows

## Confluence Integration

Documentation in this directory will be automatically synced to Confluence for team collaboration and review.
